# Retro Game Wiki

中文经典游戏百科站 — 攻略 Wiki / 模拟器资料 / 游戏数据库

## 技术栈

- **前端**: Next.js 15+ / TypeScript / Tailwind CSS / shadcn/ui
- **后端**: Next.js Server Actions + Supabase API
- **数据库**: Supabase PostgreSQL
- **部署**: Vercel + Cloudflare
