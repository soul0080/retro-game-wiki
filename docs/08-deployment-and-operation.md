# Retro Game Wiki 部署与运营方案

版本：v1.0

项目类型：

个人运营型游戏百科网站。

目标：

以最低运营成本，实现长期稳定运行，并通过广告收入实现可持续发展。

---

# 1. 总体部署架构

最终架构：

```text
用户

↓

Cloudflare CDN

↓

Vercel

↓

Next.js 网站

↓

Supabase

├ PostgreSQL数据库

├ Storage图片存储

└ Authentication


↓

AI服务

↓

爬虫服务器

```

---
