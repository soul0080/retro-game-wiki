# Retro Game Wiki 内容战略文档

版本：v1.0

项目定位：

中文经典游戏资料库、攻略 Wiki、模拟器知识平台。

核心目标：

通过长期积累高质量游戏资料，获得搜索引擎自然流量，并通过广告实现商业化。

---

# 1. 内容战略核心理念

## 1.1 不追热点，做长期资产

本网站不参与：

* 新游戏新闻竞争
* 游戏媒体速度竞争
* 游戏直播内容竞争

原因：

个人运营无法和大型游戏媒体竞争。

重点：

建立生命周期超过10年的内容。

例如：

用户5年后仍会搜索：

```
最终幻想6攻略

火焰纹章圣战系谱攻略

GBA模拟器设置

PS1经典游戏推荐
```

这些内容具有长期价值。
